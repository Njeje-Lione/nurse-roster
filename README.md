# Nurse Duty Roster System

A web-based staff scheduling system built for healthcare environments.

## Features
- Shift assignment and management
- Staff availability tracking
- Conflict detection to prevent double-booking
- Clean dashboard interface

## Getting started
```bash
git clone https://github.com/Njeje-Lione/nurse-roster.git
cd nurse-roster
pip install -r requirements.txt
python manage.py runserver
```
